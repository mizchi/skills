# node/

Skills for Node.js backend, server-side, and runtime concerns.

> Skills will be added here as they are created.

Planned:
- `node-audit` — Node.js dependency audit, CVE triage for server-side runtime (RCE/SSRF/Path Traversal risks that don't apply to browser SPAs)
- `node-performance` — profiling, event loop latency, stream backpressure
- `node-security` — environment secrets, dependency injection, SSRF, path traversal
